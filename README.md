# Tic-tae-toe

Классическая игра "Крестики-нолики" для двух игроков.

## Требования
- Python 3.11+
- Windows / macOS / Linux

## Описание
Tic Tac Toe — это игра на поле 3×3, где два игрока по очереди ставят X или O. Цель — первым выстроить три своих символа в ряд по горизонтали, вертикали или диагонали.

## Как играть
1. Игроки выбирают X или O.
2. По очереди ставят символы на пустые клетки.
3. Игра заканчивается, когда один из игроков выстроил три символа в ряд или поле полностью заполнено (ничья).

## Запуск игры
1. Скачайте готовый файл игры из [Releases](#).  
2. Убедитесь, что у вас установлен Python 3.11 или новее.  
3. Запустите игру двойным кликом по файлу `tic_tac_toe.py`
